\chapter{Introduction}
This chapter motivates per-pixel temporal modeling as an alternative to
block-based codecs. We outline the contributions of TimeFlowCodec (TFC):
\begin{itemize}
  \item Per-channel temporal regression using least squares fits (CONST, LINEAR).
  \item A simple, transparent .tfc container with per-channel payloads.
  \item Reference implementation, tests, and CLI/GUI tooling.
\end{itemize}
